###########
Discord Bot
###########

Discord Bot for general use.

========
Commands
========

* !hello

Say whats up

* !roll [d?[0-9]+]

Get a random number between 1 and the number given

* !windows

Get an inspirational message about your Operating System.

================
Planner Commands
================

* !planner register

Register for planner service

* !planner event create <event-name> [<start date>] [<end date>]

Create an event with a given name

* !planner event list

List open events

* !planner available <event-name-or-id> <date-range> [<date-range>, <date-range> ...]

Set availablity for event to given dates

* !planner event available <event-name-or-id>

Show availablities of event go-ers for an event
